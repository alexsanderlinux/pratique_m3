

HTML Semântico: Utilize as tags HTML5 adequadas para dar significado semântico aos elementos da sua página (header, nav, main, section, article, aside, footer).

CSS Externo: Crie um arquivo CSS externo para estilizar toda a página, separando a estrutura do conteúdo da apresentação visual.

Seletores: Utilize seletores CSS para aplicar estilos a elementos específicos ou grupos de elementos.

Cores, Fontes, Box-model: Aplique as propriedades CSS para definir cores, fontes, tamanhos e espaçamentos dos elementos, utilizando o modelo de caixa (box-model).

Pseudo-classes e Pseudo-elementos: Utilize pseudo-classes e pseudo-elementos para criar efeitos visuais e interações com o usuário (hover, active, before, after).

Boas Práticas: Siga as boas práticas de desenvolvimento web, como organização do código, semântica e CSS reset

Como submeter sua entrega: Envie os arquivos index.html e o arquivo CSS correspondente.
